Breve guía para escalar COMODISIMOS

- Migración a Postgres: Cambiar driver y usar migraciones (Knex/TypeORM/Prisma).
- Autenticación centralizada: OAuth2 / Identity Provider para SSO corporativo.
- Auditoría: events table y logs estructurados para trazabilidad.
- Infraestructura: contenedores Docker, despliegue en Kubernetes o App Service.
- Observabilidad: agregar métricas y trazas (Prometheus, Grafana, OpenTelemetry).
